About GTA V

Released in 2013, Grand Theft Auto V is one of the most modded games in history. The PC version has an incredibly active modding community creating everything from simple vehicle replacements to total conversion mods, custom missions, and advanced graphical enhancements.

With tools like OpenIV, Script Hook V, and CodeWalker, modders have access to nearly every aspect of the game including vehicles, weapons, character models, map locations, scripts, and audio.

Getting Started with GTA V Modding

New to GTA V modding? Start with OpenIV to learn the game's file structure. Then install Script Hook V to enable script mods. Always backup your game files before installing mods, and use a mod manager to keep things organized.

Important: Mods are for single-player only. Never use mods in GTA Online as they can result in a permanent ban.
