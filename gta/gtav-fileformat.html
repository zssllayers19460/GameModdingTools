GTA V File Formats

Complete reference guide to GTA V's archive and asset formats

Understanding GTA V File Structure

GTA V uses the RAGE Engine, which organizes game assets in RPF archives. These archives contain all game data including models, textures, scripts, audio, and configuration files. Understanding these formats is essential for effective modding.

📦 Archive Formats

.RPF

Rockstar Package File — The main archive format for GTA V. Contains all game assets compressed and organized. RPF files can be encrypted or unencrypted.

OpenIVRPF Explorer

.META / .YMT

XML-based Metadata Files — Contains game configuration data like vehicle handling, weapon stats, ped attributes, and gameplay settings.

Notepad++VS Code

.PATCH

Game Update Archives — Contains DLC content and game updates. Used for adding custom content without replacing original files.

OpenIV

🎨 Model & Mesh Formats

.YDR / .YFT

Drawable Models — YDR for static props, YFT for vehicles. Contains 3D geometry, materials, and textures.

ZModeler 3CodeWalker

.YDD

Drawable Dictionary — Collections of drawable models used for character parts, clothing, and props.

OpenIVCodeWalker

.YMAP

Map Placement Data — Defines where objects, props, and entities are placed in the game world.

CodeWalkerOpenIV

.YCD

Collision Data — Defines physics boundaries for objects and environment interaction.

CodeWalker

🎵 Texture & Audio Formats

.YTD

Texture Dictionary — Contains DDS textures for models, vehicles, and environment.

OpenIVPhotoshop (DDS)

.AWC

Audio Wave Container — Contains sound effects, radio music, dialogue, and ambient audio.

AudacityAWC Tool

.GFX

Scaleform GFX — Flash-based UI elements like menus, HUD, and in-game displays.

JPEXS Flash Decompiler

📜 Script & Configuration Formats

.YSC

Script Files — Compiled scripts for missions, gameplay logic, and AI behavior.

Script Hook VCodewalker

.GXT2

Localization Files — Contains all text strings used in the game across different languages.

OpenIV

.DAT / .INI

Configuration Files — Plain text configuration for game settings and mod loaders.

Notepad++
