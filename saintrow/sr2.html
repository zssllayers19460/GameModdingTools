ThomasJepp Saints Row Tools

Asset Extraction

A specialized utility that unpacks Saints Row's VPP_PC archive files, allowing modders to access and extract individual game assets like character models, textures, animations, and audio. This tool is the foundation for all asset-based modding in Saints Row 2—it bridges the gap between the game's compressed file format and usable asset files that creators can modify and repack.

Texture Editing

Specialized tool for extracting and editing ".peg" texture files. This is essential for any cosmetic modding, allowing you to replace in-game textures with custom versions. Works with character skins, vehicle paintjobs, clothing, and environmental textures.
