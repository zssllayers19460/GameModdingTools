About Saints Row (2022) Modding

The 2022 Saints Row reboot is a newer game with a different engine than previous titles. The modding community for this version is still growing, with tools focused primarily on texture modifications and file extraction. The newer engine provides both opportunities and challenges compared to legacy Saints Row games.

Getting Started: Start with the Texture Unpacker to modify character skins and cosmetics, then move on to the Extraction Tool for more advanced modifications. For the latest tools and community guides, visit the Saints Row 2022 forum section.
