CMESH to OBJ

Convert Saints Row 2 models to editable OBJ format for Blender

What is CMESH to OBJ?

CMESH to OBJ (cmesh2obj) converts Saints Row 2's proprietary .cmesh_pc and .smesh_pc model files to standard .OBJ format that can be edited in any 3D modeling software like Blender, Maya, or 3DS Max.

Step 1: Download and Setup

Download: cmesh2obj_0-2.rar
Source: saintsrowmods.com/forum/attachments/cmesh2obj_0-2-rar.27311

Extract to a folder. You'll need cmesh2obj.exe and optionally Python scripts for batch processing.

Step 2: Extracting Model Files

1. Use ThomasJepp Tools to extract .vpp_pc files:
- characters.vpp_pc (character models)
- vehicles.vpp_pc (vehicle models)
- weapons.vpp_pc (weapon models)
2. Look for .cmesh_pc and .smesh_pc files

Step 3: Converting to OBJ Format

# Basic conversion
cmesh2obj.exe "model.cmesh_pc" -o "model.obj"

# Convert with skeleton data
cmesh2obj.exe "character.cmesh_pc" -s "character.skl_pc" -o "character.obj"

# Batch convert
for %f in (*.cmesh_pc) do cmesh2obj.exe "%f" -o "%~nf.obj"

Step 4: Importing to Blender

  1. Open Blender, delete default cube
  2. File → Import → Wavefront (.obj)
  3. Select your converted .obj file
  4. Check "Keep Vertex Order" and "Import UVs"

Step 5: Exporting from Blender

File → Export → Wavefront (.obj)
Settings:
- Check "Write Materials"
- Enable "Triangulate Faces"
- Maintain original scale

Step 6: Converting Back to Game Format

# Use Submesh Tool to convert back
submesh_tool.exe "model.obj" -o "model_new.cmesh_pc"
🔧 Note: You'll need the Model Submesh Tool for re-importing.

Troubleshooting

  • Model invisible: Check material assignments
  • Game crashes: Verify vertex count and polygon limits
  • Wrong scale: Import original first to check scale reference
