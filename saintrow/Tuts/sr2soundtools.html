Audio Tool Suite

Complete guide to Saints Row 2 audio modding - custom radio, sound effects, and voice lines

What is the Audio Tool Suite?

The Audio Tool Suite is a collection of utilities for working with Saints Row 2 audio files. It includes UnXWB, XactExtract, UnWAV, and XactTool - everything you need to extract, edit, and repack game audio including radio stations, sound effects, music, and character voice lines.

Step 1: Download and Setup

Download the complete audio toolkit:

Download: sr2soundtools-with-unxwb.zip
Source: saintsrowmods.com/forum/attachments/sr2soundtools-with-unxwb-zip.46948/

Extract to your modding tools folder. The package includes:

  • UnXWB.exe - Extract audio from XWB banks
  • XactExtract.exe - Command-line XWB extraction
  • UnWAV.exe - Repack audio back to XWB format
  • XactTool.exe - GUI audio bank manager

Step 2: Understanding Audio Files

.xwb
XACT Wave Bank - contains audio data
.xsb
XACT Sound Bank - contains sound metadata
.xgs
XACT Global Settings

Audio is stored in audio.vpp_pc. Key banks:

  • radio.xwb - Radio station music and DJ chatter
  • sfx.xwb - Sound effects (guns, explosions, vehicles)
  • voice.xwb - Character voice lines
  • music.xwb - Background music

Step 3: Extracting Audio Files

Use ThomasJepp Tools first, then extract audio banks:

# Extract audio.vpp_pc with ThomasJepp Tools
ThomasJepp.SaintsRow.ExtractVPP.exe "audio.vpp_pc"

# Now use UnXWB to extract individual audio files
UnXWB.exe radio.xwb

# This creates a folder with all .wav files from that bank

Step 4: Working with XactExtract

Command-line extraction for batch processing:

# Basic extraction
XactExtract.exe radio.xwb

# Extract with sound names from XSB file
XactExtract.exe radio.xwb radio.xsb

# Extract specific track by index
XactExtract.exe radio.xwb -index 15 -o "my_track.wav"

# Batch extract all XWB files in folder
for %f in (*.xwb) do XactExtract.exe "%f"

Step 5: Editing Audio Files

Prepare your custom audio files:

  • Format Requirements: 44.1 kHz, 16-bit, Stereo WAV
  • Recommended Software: Audacity (free), Adobe Audition, GoldWave
  • Keep Original Length: Match duration of replaced audio for best compatibility
  • Normalize Volume: Match original audio levels to avoid distortion
🎵 Pro Tip: When creating custom radio stations, use consistent volume levels across all tracks.

Step 6: Using UnWAV to Repack

Repack edited audio back into XWB format:

# Basic repack (replaces all audio in bank)
UnWAV.exe radio.xwb custom_audio_folder\

# Repack with specific wave file
UnWAV.exe radio.xwb custom_track.wav -replace 15

# Create new XWB from scratch
UnWAV.exe -create new_radio.xwb track1.wav track2.wav track3.wav

Step 7: Custom Radio Station Guide

Create your own radio station step by step:

  1. Extract the radio bank you want to replace (e.g., The Krunch 106.66)
  2. Identify track indices for music and DJ segments
  3. Edit or replace music tracks with your own songs
  4. Optional: Record custom DJ commentary
  5. Use UnWAV to repack the radio bank
  6. Test in-game to ensure proper playback

Step 8: Voice Line Replacement

Modify character dialogue:

1. Extract voice.xwb from audio.vpp_pc
2. Use XactExtract with the XSB file to get proper names:
XactExtract.exe voice.xwb voice.xsb
3. Locate dialogue lines by character and mission
4. Edit WAV files with your custom voice recordings
5. Ensure duration matches original to avoid sync issues
6. Repack with UnWAV

Step 9: Sound Effect Modding

Replace weapon sounds, vehicle engines, etc.:

  • Weapon Sounds: sfx.xwb contains gunshots, reloads
  • Vehicle Engines: Engine sounds are in sfx.xwb
  • Environmental: Weather, ambient sounds
  • UI Sounds: Menu clicks, notifications

Step 10: Testing and Installation

Install your audio mod:

1. Place modified .xwb file in mod folder:
Saints Row 2\mods\YourMod\audio.vpp_pc\

2. Maintain original file name (e.g., radio.xwb)

3. Launch the game and test:
- Custom radio: Tune to the modified station
- Voice lines: Trigger the mission/dialogue
- Sound effects: Use weapons/drive vehicles

4. Check for audio glitches or sync issues

Troubleshooting Audio Mods

  • No sound in-game: Verify file format (44.1kHz, 16-bit WAV)
  • Crackling/distortion: Reduce volume levels, normalize audio
  • Audio out of sync: Match original track length exactly
  • Radio skips/glitches: Check track indices and XSB references
  • Game crashes on audio load: Ensure XWB file is valid, check size limits
🎧 Advanced Techniques:
  • Use XactTool GUI for easier bank management
  • Create custom radio station names and artwork
  • Add multiple sound variations for realistic effects
  • Compress large audio files to save space (use 128-192kbps)
  • Backup original banks before making changes
