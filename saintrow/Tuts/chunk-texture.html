Chunk Texture Tool

Beginner-friendly GUI tool for Saints Row 2 texture editing

What is Chunk Texture Tool?

Chunk Texture Tool is a user-friendly graphical interface for editing Saints Row 2 texture files. Unlike command-line tools, this GUI makes texture modding accessible to beginners with an intuitive drag-and-drop interface.

Step 1: Download and Installation

Get the latest version of Chunk Texture Tool:

Download: SR2ChunksTextureTool_v1.28.zip
Source: saintsrowmods.com/forum/attachments/sr2chunkstexturetool_v1-28-zip.27300

Extract to any folder and run SR2ChunksTextureTool.exe. No additional setup required.

Step 2: Understanding the Interface

Chunk Texture Tool has three main sections:

  • File Browser: Navigate to your extracted .g_peg_pc files
  • Texture Preview: View the selected texture before editing
  • Edit Panel: Import new textures, adjust settings, and export

Step 3: Opening a Texture File

  1. Click "Open File" or drag a .g_peg_pc file into the window
  2. The tool will display all textures contained in the file
  3. Select a texture from the list to preview it
  4. You can now see the texture's properties (resolution, format, mipmaps)

Step 4: Exporting for Editing

To edit a texture in external software:

1. Select the texture you want to edit
2. Click "Export" button
3. Choose DDS format (DXT5 recommended)
4. Save to your working folder
5. Edit with Photoshop, GIMP, or Paint.NET
💡 Tip: Always keep a backup of the original texture in case something goes wrong.

Step 5: Importing Edited Textures

After editing your DDS file, import it back:

  1. In Chunk Texture Tool, select the texture you want to replace
  2. Click "Import" button
  3. Browse to your edited DDS file
  4. The tool will verify compatibility (resolution, format)
  5. Click "Save" to create the modified .g_peg_pc file

Step 6: Texture Format Guide

Texture TypeRecommended FormatUsage
Character SkinsDXT5Clothing, faces, body textures
Vehicle PaintDXT1Car paint, decals without transparency
UI ElementsDXT5HUD, menus, icons with transparency
EnvironmentDXT1Buildings, terrain, props

Step 7: Common Modding Examples

  • Custom Clothing: Edit customize_item.vpp_pc textures to create unique outfits
  • Vehicle Liveries: Modify vehicle textures for custom paint jobs
  • UI Overhaul: Change HUD colors, fonts, and menu backgrounds
  • Character Tattoos: Add custom tattoos or body art

Troubleshooting

  • Import fails: Check DDS format and dimensions match original
  • Texture corrupted in-game: Verify compression format (DXT1 vs DXT5)
  • Tool crashes on large files: Extract only the texture you need first
  • Preview not showing: Install DirectX End-User Runtime
🎨 Pro Tips:
  • Use batch processing for editing multiple textures
  • Combine with PEG Tool for advanced texture manipulation
  • Test your mods frequently to catch issues early
  • Join the Discord community for feedback and help
