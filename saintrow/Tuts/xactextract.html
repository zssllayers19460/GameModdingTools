XactExtract

Command-line XWB audio extraction tool for Saints Row 2

What is XactExtract?

XactExtract is a powerful command-line utility for extracting audio files from XWB (XACT Wave Bank) containers used in Saints Row 2. It's part of the Audio Tool Suite and provides flexible options for extracting individual tracks or entire banks.

Step 1: Download and Setup

XactExtract is included in the Audio Tool Suite:

Download: sr2soundtools-with-unxwb.zip
Extract to: C:\SR2_Modding\AudioTools\

XactExtract.exe is a command-line tool - best used from Command Prompt or PowerShell.

Step 2: Basic Syntax

XactExtract.exe [XWB file] [options]
Option Description
-o <filename> Output specific track to file
-index <n> Extract specific track by index
-all Extract all tracks
-list List all tracks in the XWB
-xsb <file> Use XSB file for track names

Step 3: Listing Tracks

See what's inside an XWB file:

# Basic listing
XactExtract.exe radio.xwb -list

Step 4: Extracting a Single Track

Extract one specific audio track:

# Extract track 5 to a WAV file
XactExtract.exe radio.xwb -index 5 -o "my_track.wav"

Step 5: Extracting All Tracks

Extract every track from an XWB bank:

# Extract all tracks to current folder
XactExtract.exe radio.xwb -all

Step 6: Using XSB Files

Get proper track names from the XSB sound bank:

# Extract with names from XSB
XactExtract.exe radio.xwb radio.xsb -all
💡 Tip: Always use XSB files when available for better organization.

Step 7: Batch Processing

Extract multiple XWB files at once:

# Batch script example (Windows batch file)
@echo off
for %%f in (*.xwb) do (
echo Processing %%f
XactExtract.exe "%%f" -all
)

Troubleshooting

  • "File not found" error: Check file path, use full path if needed
  • Corrupted WAV output: Try different extraction method
  • Missing tracks: Check if XSB file is needed for proper indexing
  • Tool crashes: Run as administrator, check file permissions
