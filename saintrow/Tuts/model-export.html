Model Export Tool

Extract and edit Saints Row 2 3D models for Blender, Maya, and 3DS Max

What is Model Export Tool?

Model Export Tool (cmesh2obj) converts Saints Row 2's proprietary .cmesh_pc and .smesh_pc model files to standard .OBJ format that can be edited in any 3D modeling software. This is essential for creating custom characters, vehicles, and weapons.

Step 1: Understanding Model Files

.cmesh_pc

Complex mesh files - characters, vehicles, detailed props

.smesh_pc

Simple mesh files - low-poly props, environment objects

.skl_pc

Skeleton files - bone structure for animated models

Step 2: Download and Setup

Get the latest version of Model Export Tool:

Download: cmesh2obj_0-2.rar
Source: saintsrowmods.com/forum/attachments/cmesh2obj_0-2-rar.27311

Extract to a folder. You'll need:

  • cmesh2obj.exe (command-line converter)
  • Python scripts for batch processing (optional)
  • Documentation and examples

Step 3: Extracting Model Files

First, you need to extract the model files from the game archives:

1. Use ThomasJepp Tools to extract the relevant .vpp_pc files:
- characters.vpp_pc (character models)
- vehicles.vpp_pc (vehicle models)
- weapons.vpp_pc (weapon models)
- props.vpp_pc (environment objects)

2. Look for .cmesh_pc and .smesh_pc files in the extracted folders

Step 4: Converting to OBJ Format

Use cmesh2obj to convert game models to OBJ:

# Basic conversion
cmesh2obj.exe "model.cmesh_pc" -o "model.obj"

# Convert with skeleton data
cmesh2obj.exe "character.cmesh_pc" -s "character.skl_pc" -o "character.obj"

# Batch convert all files in folder
for %f in (*.cmesh_pc) do cmesh2obj.exe "%f" -o "%~nf.obj"
⚠️ Important: Keep the .obj and .mtl files together - they contain material data.

Step 5: Importing to Blender

How to import your models into Blender:

  1. Open Blender and delete the default cube
  2. File → Import → Wavefront (.obj)
  3. Select your converted .obj file
  4. Check "Keep Vertex Order" and "Import UVs" options
  5. The model will appear with its textures mapped

Step 6: Editing Models

Common modifications you can make:

  • Mesh Editing: Reshape, add or remove geometry
  • UV Mapping: Adjust texture coordinates
  • Material Changes: Modify colors and properties
  • Rigging: Create custom skeletons for animations
💡 Tip: Maintain vertex count and hierarchy structure to avoid game crashes.

Step 7: Exporting from Blender

After editing, export back to OBJ format:

File → Export → Wavefront (.obj)
Important settings:
- Check "Write Materials"
- Keep "Objects as OBJ Objects"
- Enable "Triangulate Faces"
- Maintain original scale (1.0)

Step 8: Converting Back to Game Format

Use Submesh Tool to convert OBJ back to .cmesh_pc:

# Convert OBJ back to game format
submesh_tool.exe "model.obj" -o "model_new.cmesh_pc"

# If you modified the skeleton
submesh_tool.exe "model.obj" -s "original.skl_pc" -o "model_new.cmesh_pc"
🔧 Note: The Submesh Tool is required for re-importing models back into the game.

Step 9: Testing Your Model

To test your custom model in-game:

1. Place the modified .cmesh_pc file in your mod folder
2. Maintain original file structure:
Saints Row 2\mods\YourMod\vehicles.vpp_pc\custom_car.cmesh_pc
3. Launch the game and spawn your model
4. Check for any visual glitches or crashes

Common Model Mods

  • Custom Vehicles: Import real cars or create unique designs
  • Character Models: Create new outfits, hairstyles, or body types
  • Weapon Replacements: Add realistic guns or sci-fi weapons
  • Environment Props: Add new buildings, signs, or decorations

Troubleshooting

  • Model invisible in-game: Check material assignments and UV mapping
  • Game crashes on load: Verify vertex count and polygon limits
  • Textures missing: Ensure .mtl file references correct texture paths
  • Wrong scale: Import original model first to check scale reference
🎨 Advanced Tips:
  • Use reference models to maintain proper scale and proportions
  • Keep LOD (Level of Detail) versions for better performance
  • Test with low-poly versions first before adding detail
  • Join the modding Discord for advanced rigging help
