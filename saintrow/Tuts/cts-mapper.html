CTS Mapper Tool

Edit map locations, spawn points, and mission markers in Saints Row 2

What is CTS Mapper?

CTS Mapper is a powerful tool for editing Saints Row 2's map data. It allows you to view and modify map images, spawn points, mission markers, activity locations, and other world data that defines the game environment.

Step 1: Download and Setup

Get the latest version of CTS Mapper:

Download: SR2CTSMapper2-01.zip
Source: saintsrowmods.com/forum/attachments/sr2ctsmapper2-01-zip.25577

Extract the ZIP and run SR2CTSMapper.exe. The tool requires .NET Framework 4.5 or higher.

Step 2: Loading Map Data

To begin editing, you need to load the map files:

  1. First, extract the map.vpp_pc file using ThomasJepp Tools
  2. Look for .cts files (CTS = Coordinate System)
  3. In CTS Mapper, click "File" → "Open" and select the .cts file
  4. The map will load showing all markers and zones

Step 3: Understanding Map Elements

CTS Mapper displays several types of map data:

  • Red Markers: Mission start locations
  • Blue Markers: Activity points (insurance fraud, mayhem, etc.)
  • Green Markers: Store and shop locations
  • Yellow Markers: Safehouse and garage locations
  • Purple Zones: Gang territory boundaries

Step 4: Viewing Coordinates

Each marker has X, Y, Z coordinates:

Example format: X: 1234.56, Y: 7890.12, Z: 45.67
X = East/West position
Y = North/South position
Z = Height (ground level, rooftop, etc.)

Click any marker to view and edit its coordinates in the properties panel.

Step 5: Editing Spawn Points

To modify where NPCs or vehicles spawn:

  1. Select a spawn point marker from the list
  2. Enter new X, Y, Z coordinates in the properties panel
  3. Adjust rotation if needed (facing direction)
  4. Click "Apply" to save changes
💡 Tip: Use in-game coordinates (enable debug mode) to find exact positions.

Step 6: Adding New Markers

Create custom locations for missions or activities:

1. Right-click anywhere on the map
2. Select "Add New Marker"
3. Choose marker type (Mission, Activity, Shop, etc.)
4. Enter coordinates and any additional parameters
5. Save the modified .cts file

Step 7: Modifying Activity Zones

Activities like Insurance Fraud have defined areas:

  • Select an activity zone from the list
  • Adjust the zone boundaries by dragging corner points
  • Change activity parameters (time limit, score targets)
  • Preview changes in the 3D view

Step 8: Mission Flow Editing

CTS Mapper can also modify mission trigger points:

  • Find the mission start marker you want to edit
  • Check the "Mission Trigger" properties
  • Edit the mission ID to change which mission starts
  • Add or remove requirements (gang reputation, completed missions)

Step 9: Saving and Testing

After editing, save your changes:

1. File → Save or Save As
2. Place the modified .cts file in your mod folder:
Saints Row 2\mods\YourMod\map.vpp_pc\
3. Launch the game to test changes

Common Mods with CTS Mapper

  • New Mission Locations: Add custom mission start points
  • Relocated Activities: Move Insurance Fraud to better locations
  • Custom Gang Territories: Redefine gang control areas
  • Safehouse Additions: Add new crib locations
  • Fast Travel Points: Create custom teleport locations

Troubleshooting

  • Map not loading: Ensure .cts files are extracted correctly
  • Markers not appearing: Check coordinates are within map bounds
  • Game crashes: Verify marker types and parameters are valid
  • Changes not showing: Confirm mod folder structure and load order
📝 Advanced Techniques:
  • Export coordinates to text files for documentation
  • Use the coordinate calculator for precise positioning
  • Combine with script editing for full mission creation
  • Create multiple versions to test different layouts
