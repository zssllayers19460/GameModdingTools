FPS Unlocker & Performance Guide

Unlock frame rates and optimize Saints Row 2 for modern hardware

What is FPS Unlocker?

Saints Row 2 PC was originally capped at 30 FPS. The FPS Unlocker removes this limitation, allowing the game to run at high frame rates on modern monitors. This tutorial covers the unlocker and additional performance tweaks for the best experience.

⚠️ Important Warning: High frame rates can affect game physics and mission triggers. Always test thoroughly and consider capping at 60 FPS for stability.

Step 1: Download FPS Unlocker

Get the latest version compatible with your game version:

Download: gog_sr2_fps_unlocker.zip (for GOG version)
Source: saintsrowmods.com/forum/attachments/gog_sr2_fps_unlocker-zip.26084

Note: Different versions exist for Steam, GOG, and retail versions.

Step 2: Installation

Install the FPS Unlocker:

  1. Extract the FPS unlocker files
  2. Copy the .dll and .exe files to your Saints Row 2 installation folder
  3. Launch the game through the unlocker's .exe (not the original)
  4. For Steam: Add the unlocker .exe as a non-Steam game or use launch options

Step 3: Configuration Options

Edit the config file to customize FPS limits:

# FPSUnlocker.ini settings
[Settings]
FPSLimit = 60 # Set your desired limit (0 = unlimited)
VSync = 1 # Enable/disable VSync
Borderless = 0 # Borderless window mode
ResolutionScale = 1.0 # Render scale (higher = better quality)

# Recommended settings for different monitors:
# 60Hz monitor: FPSLimit = 60
# 144Hz monitor: FPSLimit = 144 or 120
# 240Hz monitor: FPSLimit = 240 or 120

Step 4: Display & Graphics Tweaks

Additional display.cfg file modifications:

# Located in Documents\My Games\Saints Row 2\display.cfg

[Display]
FullscreenWidth = 1920
FullscreenHeight = 1080
WindowWidth = 1920
WindowHeight = 1080

[Graphics]
ShadowQuality = 2 # 0-2 (higher = better shadows)
ReflectionQuality = 2 # 0-2
DrawDistance = 1000 # Increase for less pop-in
DetailDistance = 800 # Object detail distance
PostProcessing = 1 # Bloom, motion blur, etc.

Step 5: GPU Compatibility Fixes

For modern graphics cards (RTX, RX series):

  • NVIDIA Users: Force V-Sync through NVIDIA Control Panel
  • AMD Users: Use Radeon Software to set frame rate target
  • Intel Graphics: May require additional .dll patches
💡 Tip: Disable in-game V-Sync and use GPU control panel for best results.

Step 6: Audio Sync Fixes

High frame rates can cause audio desync:

# Force audio to use system clock instead of game clock
[Audio]
UseSystemClock = 1
AudioLatency = 128 # Adjust if audio stutters

# Alternative: Use external audio driver (ASIO) for low latency

Step 7: Memory and Stability

Large Address Aware (LAA) patch for better memory usage:

1. Download Large Address Aware tool
2. Run LAA and select SR2_pc.exe
3. Check "Large Address Aware" flag
4. Apply and close
5. This allows the game to use up to 4GB RAM (instead of 2GB)

Step 8: Mod Compatibility

Ensure your mods work with FPS unlocker:

  • Gentlemen of the Row: Fully compatible, includes performance tweaks
  • Custom Missions: May need script adjustments for frame-dependent triggers
  • Vehicle Mods: Physics may behave differently at high FPS
  • Texture Mods: No conflicts - works with all texture mods

Step 9: Troubleshooting Common Issues

  • Game crashes on launch: Run as administrator, disable overlays (Discord, Steam)
  • Stuttering: Cap FPS to refresh rate, enable triple buffering
  • Physics glitches: Reduce FPS cap to 60, use Gentlemen of the Row fixes
  • Mission triggers not working: Temporarily cap FPS for specific missions
  • Input lag: Disable V-Sync, use exclusive fullscreen mode

Step 10: Benchmarking

Test your performance:

1. Use MSI Afterburner or FRAPS to monitor FPS
2. Test in different areas: driving fast, crowded areas, explosions
3. Record stable FPS minimums
4. Adjust settings until stable 60 FPS or higher
5. Save your optimized config file
🎮 Recommended Settings:
  • Low-end PC: 30 FPS cap, lowered draw distance, disabled shadows
  • Mid-range PC: 60 FPS cap, medium shadows, standard draw distance
  • High-end PC: 120-144 FPS, max settings, increased draw distance
  • Ultra-wide monitors: Use Flawless Widescreen for proper FOV
