ThomasJepp Saints Row Tools

The foundation of Saints Row 2 modding - extracting game assets

What are ThomasJepp Tools?

ThomasJepp Tools are the essential utilities for extracting Saints Row 2's VPP_PC archive files. Created by community developer Thomas Jepp, these tools unlock all game assets including textures, models, audio, and scripts for modification.

Step 1: Download and Setup

Download the latest version of ThomasJepp Tools:

Download: ThomasJepp.SaintsRow-rev121.zip
Source: minimaul.saintsrowmods.com/files/tools/releases/

Extract the ZIP to a folder (e.g., C:\SR2_Modding\ThomasJepp). No installation required.

Step 2: Understanding VPP_PC Files

Saints Row 2 stores game data in .vpp_pc archive files. Common archives include:

  • misc.vpp_pc - Textures, UI elements, environment data
  • customize_item.vpp_pc - Clothing, character customization
  • audio.vpp_pc - Sound effects, music, voice lines
  • vehicles.vpp_pc - Vehicle models and data
  • weapons.vpp_pc - Weapon models and textures

Step 3: Extracting Archives

Using the command-line tool to extract VPP_PC files:

# Basic extraction command
ThomasJepp.SaintsRow.ExtractVPP.exe "C:\Game\Saints Row 2\misc.vpp_pc"

# Extract to specific folder
ThomasJepp.SaintsRow.ExtractVPP.exe "misc.vpp_pc" -o "C:\Extracted\misc"

# Batch extract multiple files
for %f in (*.vpp_pc) do ThomasJepp.SaintsRow.ExtractVPP.exe "%f"
⚠️ Important: Always work on copies of the original files. Never modify original game files directly.

Step 4: Exploring Extracted Files

After extraction, you'll find various file types:

  • .g_peg_pc - Texture files (use PEG Tool)
  • .cmesh_pc / .smesh_pc - 3D model files (use Model Export Tool)
  • .xwb - Audio banks (use Audio Tool Suite)
  • .cts - Map and mission data (use CTS Mapper)
  • .xtbl - Game data tables (editable with text editor)

Step 5: Creating Mod Folders

To create a mod without overwriting game files:

# Create mod folder structure
Saints Row 2\
└── mods\
└── MyFirstMod\
└── misc.vpp_pc\
└── (extracted files go here)

The game automatically loads mods from this folder structure, overriding original files.

Step 6: Repacking for Distribution

When sharing your mod, repack into .vpp_pc format:

# Repack command
ThomasJepp.SaintsRow.BuildVPP.exe "MyMod" -o "MyMod.vpp_pc"

# This creates a distributable .vpp_pc file

Common Issues & Solutions

  • Extraction fails: Run as administrator, check file permissions
  • Missing files: Ensure you extracted the correct .vpp_pc
  • Game not loading mod: Verify folder structure and file paths
  • Command not recognized: Use full path or add to PATH environment
🔧 Advanced Techniques:
  • Use the GUI version for easier file browsing
  • Create batch scripts for repetitive extraction tasks
  • Combine with other tools for complete modding workflow
