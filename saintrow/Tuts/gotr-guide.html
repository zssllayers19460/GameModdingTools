Gentlemen of the Row

The essential Saints Row 2 super-mod - complete installation and customization guide

What is Gentlemen of the Row?

Gentlemen of the Row (GotR) is the definitive super-mod for Saints Row 2, created by the legendary IdolNinja. It fixes hundreds of bugs, restores cut content, adds new clothing, weapons, vehicles, missions, and countless gameplay tweaks. It's considered mandatory for the best Saints Row 2 experience on PC.

Memory optimizations, loading improvements, crash fixes

Step 2: Pre-Installation Preparation

Before installing, make sure you have:

  • A clean installation of Saints Row 2 (Steam or GOG version)
  • Backup of your save files (located in Documents\My Games\Saints Row 2)
  • Backup of your game installation (optional but recommended)
  • At least 2GB of free disk space

Step 3: Extract the Mod Files

Extract the downloaded archive:

1. Right-click the .rar file and select "Extract Here"
2. Navigate to the extracted folder
3. You should see:
- Gentlemen_of_the_Row_Saints_Row_2_Super_Mod_v1.9.2/
  ├── build/ (mod building tool)
  ├── optional_mod_stuff/ (additional options)
  ├── README.txt
  └── Run_Gentlemen_of_the_Row.bat

Step 4: Running the Mod Builder

Build your customized version of GotR:

1. Double-click Run_Gentlemen_of_the_Row.bat
2. The tool will open with a menu of options
3. Select your game type (Steam, GOG, or Retail)
4. Choose your installation folder
5. Select optional features to include
6. Press Enter to build the mod
7. Wait for completion (usually 2-5 minutes)

Step 5: Customization Options

GotR offers extensive customization:

  • Cheats: Unlock all cheats, add new cheats
  • Clothing: New outfits, accessories, customization options
  • Weapons: Restored weapons, new weapons, weapon stats
  • Vehicles: Restored vehicles, custom handling, new colors
  • Gameplay: Difficulty tweaks, activity changes, money adjustments
  • Visuals: ENB options, texture improvements, UI changes

Step 6: Installing the Mod

After building, install the mod:

1. The builder creates a "MY_CUSTOM_GOTR" folder
2. Copy the contents of "MY_CUSTOM_GOTR" to your Saints Row 2 installation folder
3. When prompted, overwrite existing files
4. The mod is now installed and ready to play
⚠️ Note: Always launch the game through the regular executable. GotR modifies game files directly.

Step 7: Post-Installation Setup

Configure additional settings:

  • Launch the game once to generate new config files
  • Adjust display.cfg for optimal performance
  • Enable/disable specific GotR features via in-game menu
  • Consider installing FPS Unlocker for better performance

Step 8: Verifying Installation

Check if GotR is working correctly:

1. Start a new game or load a save
2. Look for new clothing options in stores
3. Check for new vehicles in garages
4. Verify restored missions are available
5. Test cheat codes to see if new cheats work

Step 9: Adding Other Mods

GotR is compatible with many other mods:

  • Texture Mods: Use PEG Tool mods alongside GotR
  • Vehicle Mods: CVTF Helper mods work with GotR
  • Audio Mods: Custom radio and sound mods
  • Install Order: Install GotR first, then other mods

Step 10: Troubleshooting

  • Game crashes on launch: Verify installation path, run as administrator
  • Missing features: Rebuild with all options selected
  • Performance issues: Install FPS Unlocker, reduce graphical settings
  • Save game corruption: Load older save, verify game files
  • Mod conflicts: Remove other mods, reinstall GotR

Step 11: Uninstalling

To remove GotR and restore original game:

1. Verify game files through Steam/GOG
2. This will restore all original files
3. Delete any leftover mod files in the game folder
4. Reinstall if you want to start fresh
