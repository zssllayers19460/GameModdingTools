UnWAV

Repack audio files back into Saints Row 2 XWB format

What is UnWAV?

UnWAV is the companion tool to XactExtract - it takes edited WAV files and repacks them into XWB (XACT Wave Bank) format that Saints Row 2 can read. This completes the audio modding workflow.

Step 1: Audio Format Requirements

Required Format:
- Sample Rate: 44.1 kHz (44100 Hz)
- Bit Depth: 16-bit
- Channels: Stereo (or Mono for voice/sfx)
- Format: PCM WAV
🎵 Important: Use Audacity to convert your audio to the correct format.

Step 2: Basic Repack Command

Replace all audio in an XWB bank:

UnWAV.exe radio.xwb custom_audio_folder\

Step 3: Replacing Individual Tracks

Replace specific tracks while keeping others:

# Replace track 5 only
UnWAV.exe radio.xwb custom_track.wav -replace 5

Step 4: Creating New XWB Banks

Build an entirely new XWB file from scratch:

UnWAV.exe -create my_radio.xwb song1.wav song2.wav song3.wav

Step 5: Custom Radio Station

  1. Extract original radio.xwb to see track structure
  2. Identify music and DJ commentary tracks
  3. Prepare your custom songs (44.1kHz, 16-bit, WAV)
  4. Replace tracks with UnWAV
  5. Test in-game

Step 6: Voice Line Replacement

1. Extract voice.xwb with XactExtract and XSB
2. Find the dialogue line to replace
3. Record custom line (match length and cadence)
4. UnWAV.exe voice.xwb custom_line.wav -replace [index]

Step 7: Testing Your Audio Mod

1. Place modified .xwb file in:
Saints Row 2\mods\YourMod\audio.vpp_pc\
2. Launch the game and test

Troubleshooting

  • No sound: Verify format (44.1kHz, 16-bit, WAV)
  • Clicking/popping: Normalize audio before repacking
  • Game crashes: Ensure track count matches original
