PEG Tool Tutorial

Complete guide to extracting and editing Saints Row 2 texture files

What is PEG Tool?

PEG Tool is an essential utility for Saints Row 2 texture modding. It allows you to extract, view, and edit .peg texture files used throughout the game for character skins, vehicle paintjobs, clothing, and environment textures.

Step 1: Download and Installation

First, download the latest version of PEG Tool from the Saints Row Mods community:

Download: PEG Tool v1.2.8
Source: saintsrowmods.com/forum/attachments/sr2pegtool_v128-zip.27301

No installation required - simply extract the ZIP file to a folder of your choice (e.g., C:\SR2_Modding\PEGTool).

Step 2: Extracting Game Textures

To begin modding, you need to extract the game's texture files:

  1. Navigate to your Saints Row 2 installation folder (usually C:\Program Files (x86)\Steam\steamapps\common\Saints Row 2)
  2. Use ThomasJepp Tools to extract misc.vpp_pc and customize_item.vpp_pc
  3. Look for .g_peg_pc files - these are your texture files
  4. Open PEG Tool and drag a .g_peg_pc file onto the executable
  5. The tool will extract all textures as DDS files to the same folder
💡 Pro Tip: Create a dedicated "extracted" folder to keep your original files organized.

Step 3: Editing Textures

Once extracted, you can edit the DDS files using image editing software:

  • Required software: Photoshop with NVIDIA DDS plugin, GIMP with DDS plugin, or Paint.NET
  • Save settings: Always save as DDS with DXT5 compression for most textures
  • Alpha channels: Some textures use alpha for transparency - preserve these
  • Resolution: Keep original dimensions to avoid game crashes

Step 4: Repacking Textures

After editing, you need to repack the textures back into game format:

# Command line usage
PEGTool.exe your_texture.g_peg_pc -repack

# Or use the GUI version:
- Select "Repack" mode
- Choose your edited DDS files
- Specify output location

The tool will create a new .g_peg_pc file ready for the game.

Step 5: Testing Your Mod

To test your texture mod in-game:

  1. Place your modified .g_peg_pc file in the appropriate folder structure
  2. Create a mod folder: Saints Row 2\mods\YourModName\
  3. Maintain the original folder structure (e.g., if the file came from misc.vpp_pc, put it in mods\YourModName\misc.vpp_pc\)
  4. Launch the game and check if your textures appear correctly

Common Issues & Solutions

  • Texture not showing: Check file path and ensure DXT5 compression
  • Game crashes: Verify texture dimensions match original
  • Corrupted colors: Use correct DDS format (DXT1 for opaque, DXT5 for alpha)
  • Tool not responding: Run as administrator or check file permissions
🎨 Advanced Tips:
  • Combine with Chunk Texture Tool for easier management
  • Use batch processing for multiple textures
  • Backup original files before modifying
  • Join the community at saintsrowmods.com for help
