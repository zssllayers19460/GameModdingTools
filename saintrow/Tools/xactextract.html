What is XactExtract?

XactExtract is a powerful command-line utility designed specifically for extracting audio from XWB (XACT Wave Bank) files used by Saints Row 2 and other XACT-based games. This tool converts XWB audio banks into individual WAV files, giving you access to all in-game sounds — from radio music and character voice lines to weapon sounds, vehicle noises, and ambient audio effects.

Created by IdolNinja, XactExtract is an essential part of the Saints Row 2 audio modding pipeline. It provides the first step in creating custom audio mods, allowing you to extract, edit, and eventually repack your own sound files back into the game.

How to Use XactExtract

1

Extract Game Archives

Use ThomasJepp Tools to extract the XWB audio files from the game's VPP_PC archives.

2

Open Command Prompt

Navigate to the folder containing XactExtract and your XWB files using Command Prompt or PowerShell.

3

Run Extraction

Type xactextract.exe yourfile.xwb to extract all audio from a single XWB file.

4

Batch Process

Create a batch file to extract multiple XWB files automatically.

5

Edit Your Audio

Use Audacity, Adobe Audition, or your preferred DAW to edit the extracted WAV files.

📝 Example Usage:

xactextract.exe radio_music.xwb xactextract.exe --output "C:\AudioExtracts" voices.xwb for %f in (*.xwb) do xactextract.exe "%f"

The extracted WAV files will be saved in the same folder as the source XWB file by default.

💡 Batch Processing Tip

Create a batch file (.bat) with for %%f in (*.xwb) do xactextract.exe "%%f" to extract all XWB files in a folder automatically.

Finding XWB Files in Saints Row 2

Saints Row 2 stores its audio in XWB (XACT Wave Bank) files, which are contained within the game's VPP_PC archives. Here's where to find them:

  • Radio Music: Located in radio related VPP_PC files (e.g., radio_music.vpp_pc)
  • Voice Lines: Found in character-specific VPP files (player_voices.vpp_pc, etc.)
  • Sound Effects: Stored in sound_effects.vpp_pc and similar archives
  • Mission Audio: Located in mission-specific VPP archives
XactExtract requires XWB files extracted from VPP_PC archives using ThomasJepp Saints Row Tools first.

Audio Editing After Extraction

Once you've extracted your audio with XactExtract, you'll have standard WAV files ready for editing. Recommended tools:

  • Audacity: Free, open-source audio editor — perfect for beginners
  • Adobe Audition: Professional-grade audio editing software
  • Reaper: Affordable DAW with powerful audio editing features
  • FL Studio / Ableton: For music production and complex audio manipulation

🎧 Audio Editing Tips

When editing audio for Saints Row 2, maintain the original sample rate (usually 22,050 Hz or 44,100 Hz) and bit depth (16-bit) to ensure compatibility. Always keep a backup of the original extracted files before editing.

Troubleshooting

XactExtract won't process the XWB file

Fix: Ensure the XWB file is not corrupted and was properly extracted from the VPP_PC archive. Try re-extracting with ThomasJepp Tools.

Extracted WAV files are silent or distorted

Fix: Some XWB files use different audio formats. The file may still be playable — test in a media player. If issues persist, the file may be encrypted or in a different format.

Command line closes immediately

Fix: Run Command Prompt first, then navigate to the XactExtract folder. Don't double-click the EXE directly — it's a command-line tool.

Extraction produces no files

Fix: Check that you have write permissions in the folder. Try running Command Prompt as Administrator.

Pro Tips for Audio Modding

🎧 Organize Your Extracts

Create separate folders for different audio types (Radio, Voices, Effects) to keep your modding project organized.

📝 Document File Names

Keep a spreadsheet of which audio files correspond to which in-game sounds — this makes editing much easier.

🔊 Volume Matching

When creating custom audio, match the volume levels of the original game files for a seamless experience.

💾 Backup Originals

Always keep copies of the original extracted WAV files before editing. You'll need them if something goes wrong.

⚡ Test Incrementally

Replace one audio file at a time and test in-game to identify issues quickly.

🎵 Custom Radio Stations

Radio tracks are the easiest to mod — extract, replace with your own music, repack with UnWAV.
