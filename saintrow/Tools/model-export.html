What is the Model Export Tool?

The Model Export Tool (cmesh2obj.exe) is a specialized utility that extracts 3D models from Saints Row 2 and converts them to the industry-standard .obj format. Created by community developer Masamaru, this tool works with the game's proprietary model formats — .cmesh_pc (character models), .smesh_pc (static meshes/environment), and .car_pc (vehicle models).

Using this tool is incredibly simple: just drag and drop any compatible model file onto the executable, and it will automatically generate an OBJ file in the same folder. This makes it accessible to both beginners and advanced 3D artists looking to create custom character models, vehicle mods, or environmental changes.

"Here is tool I wrote to export SR2 models to OBJ. Works with cmesh, smesh and car formats. To use it just drag and drop cmesh_pc, smesh_pc or car_pc file on exe and it would give you OBJ in same folder."

— Masamaru, Creator of Model Export Tool

Where to Find Models in the Game

After extracting VPP_PC archives with ThomasJepp Tools, you'll find model files in these locations:

  • Character Models (CMESH): characters/ folder — includes player, NPCs, gang members
  • Vehicle Models (CAR_PC): vehicles/ folder — all drivable vehicles
  • Weapon Models: weapons/ folder — weapons and equipment
  • Environment Models (SMESH): world/ and buildings/ folders
  • Prop Models: props/ folder — small world objects and decorations

🔍 Finding Specific Models

Model filenames often indicate what they are. Look for descriptive names like "player_female", "cop_male", "sports_car", etc. Extract a few to explore what's available.

Pro Tips for Model Editing

📐 Maintain Scale

When editing models, keep the original scale and proportions to ensure proper in-game placement.

🎨 Preserve UV Maps

Don't modify UV coordinates unless you're also updating textures. UV maps determine how textures wrap around the model.

🔍 Use Reference Models

Keep an unmodified version of the original model as reference while editing.

⚡ Test Incrementally

Make small changes and test frequently to identify issues early.

📁 Backup Originals

Always keep backups of original .cmesh_pc files before replacing.

🎯 Polygon Count

Be mindful of polygon counts — very high-poly models can impact game performance.

Troubleshooting

Tool doesn't create an OBJ file

Fix: Ensure the file is actually a .cmesh_pc, .smesh_pc, or .car_pc file. Check that the file isn't corrupted. Try re-extracting from VPP_PC.

OBJ won't open in 3D software

Fix: Some 3D software requires specific import settings. Try importing as "Wavefront OBJ" and ensure you have UV maps enabled.

Model looks distorted after export

Fix: The tool preserves original model structure. Check your 3D software's import settings. Some models may have scaling issues that need adjustment.

Can't find specific model files

Fix: Some models are in different VPP_PC archives. Use ThomasJepp Tools to explore all archives. Check the "patch" folder for updated models.

Tool crashes when dragging files

Fix: Make sure you're using a compatible file format. Try running the tool from Command Prompt to see error messages.
