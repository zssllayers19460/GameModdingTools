What is Gentlemen of the Row?

Gentlemen of the Row (GotR) is the iconic super-mod for Saints Row 2, created by the legendary IdolNinja. It's a massive compilation that transforms the PC version into the definitive way to play. The mod fixes hundreds of bugs, restores cut content originally scrapped during development, adds dozens of new clothing items, weapons, vehicles, and even new missions and activities. It also includes the essential community patch that makes Saints Row 2 stable on modern hardware.

For years, GotR has been considered mandatory for any player or modder. It's not just a collection of tweaks — it's a complete overhaul that respects the original game while polishing it to a mirror shine. Whether you're playing for the first time or returning to Stilwater, Gentlemen of the Row is the foundation everything else is built upon.

"Gentlemen of the Row was created to give Saints Row 2 the love it deserved on PC. It brings together fixes, restored content, and community contributions into one seamless package."

— IdolNinja (Mike Watson), Creator of GotR

How to Install Gentlemen of the Row

1

Download GotR

Get the latest version (v1.9.2) from the official Saints Row Mods page.

2

Extract the Archive

Use 7-Zip or WinRAR to extract the contents. You'll get a folder with the installer batch file.

3

Run the Installer

Double-click create_gentlemen_of_the_row.bat and follow the prompts. Select your Saints Row 2 installation folder.

4

Customize Your Build

Choose optional modules: new weapons, vehicles, gameplay tweaks, and visual enhancements.

5

Launch & Enjoy

Run Saints Row 2 normally. Gentlemen of the Row will be active.

⚠️ Important Note

Gentlemen of the Row is designed for a clean installation of Saints Row 2. Remove any other mods before installing to avoid conflicts. Backup your save files.

Troubleshooting Common Issues

Game crashes on startup after install

Fix: Ensure you selected the correct Saints Row 2 folder. Remove other mods and reinstall.

Missing textures / pink models

Fix: Re-download the archive, re-extract with 7-Zip, and run the installer again.

New weapons or vehicles not showing

Fix: Run the installer again and ensure the modules are enabled. Start a new game or visit a shop.

Audio / radio issues

Fix: Verify game files then reinstall GotR. Make sure you have the latest DirectX redistributables.
