What is ThomasJepp Saints Row Tools?

ThomasJepp Saints Row Tools is a specialized utility designed to unpack and repack Saints Row's proprietary VPP_PC archive files. VPP_PC files are compressed containers that hold the bulk of the game's assets—everything from 3D character models and vehicle meshes to textures, animations, sound effects, dialogue, and gameplay data.

This tool is the foundation of all asset-based modding in Saints Row 2. Without it, modders would have no way to access the game's internal file structures, making cosmetic changes, model swaps, texture replacements, and major gameplay modifications impossible. It bridges the gap between the game's compressed, optimized format and human-readable asset files that can be edited in external applications.

Core Function

Decompresses VPP_PC archives → Extracts individual assets → Allows editing → Repacks modified files back into game format

Troubleshooting

Game crashes on load

Fix: Verify edited assets are valid. Re-export and repack if necessary.

Modified assets don't appear

Fix: Double-check file paths match original. Ensure VPP_PC is in correct directory.

Missing textures (pink/magenta)

Fix: Verify texture file names match exactly. Re-export and repack affected assets.

Tool won't open VPP_PC

Fix: Restore from backup. Ensure using correct tool version for your game version.
