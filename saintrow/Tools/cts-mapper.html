What is CTS Mapper?

CTS Mapper is a specialized tool for viewing and editing Saints Row 2's world map. CTS stands for "Common Terrain Streamer" - it's the system Saints Row 2 uses to manage the game world, locations, spawn points, mission markers, and other world entities.

The CTS Mapper gives you a bird's-eye view of Stilwater (the game's city) and allows you to:

  • View the complete game map with all buildings and streets
  • Edit spawn locations for custom missions
  • Place and modify mission markers and objectives
  • Adjust trigger zones and world properties
  • Create custom world modifications

Core Function

View 2D map of Stilwater → Locate mission spawn points and markers → Edit positions and properties → Create custom world layouts and mission starts

CTS Mapper Workflow

Here's how to use CTS Mapper to create and modify world locations:

1

Extract Game Files

Use ThomasJepp Saints Row Tools to extract the game's VPP_PC files. You need to access the CTS data files to edit them with CTS Mapper.

2

Launch CTS Mapper

Open the CTS Mapper application. It will present you with options to load map data from your extracted game files.

Tip: Make sure your extracted files are from a clean, unmodified copy of Saints Row 2.

3

Load the Stilwater Map

Load the CTS map data for Stilwater. The tool will display a 2D overhead view of the entire game world.

Understanding the View: The map shows the entire city from above. Darker areas are streets/water, lighter areas are buildings and terrain.

4

Locate Spawn Points & Markers

Use the tool to find mission spawn points, objective markers, and other important locations. The tool shows these as icons or highlights on the map.

  • Red dots = Mission spawns
  • Yellow flags = Objective markers
  • Green circles = Safe houses
  • Blue squares = Activity locations
5

Select Location to Edit

Click on a marker, spawn point, or location you want to edit. The tool will show you its current properties including coordinates, rotation, and type.

X: 150.5 | Y: 200.3 | Z: 15.0 | Rotation: 45°
6

Modify Location Properties

Edit the location's properties in the properties panel. You can change:

  • Position (X, Y, Z): Move the location to a new spot
  • Rotation: Change facing direction
  • Type: Change what kind of location it is
  • Name: Give it a custom label
  • Linked Events: Attach mission triggers
7

Create New Locations (Optional)

Add entirely new markers, spawn points, or zones to create custom mission locations. Right-click on the map and select "Add Location".

Pro Tip: Place locations on streets or open areas, not inside buildings, to avoid clipping issues.

8

Save Your Changes

Save the modified CTS data back to your working directory. CTS Mapper will export the modified map file.

9

Repack and Test

Use ThomasJepp Tools to repack your modified CTS data back into the game's VPP_PC files. Launch the game to test your map changes.

Understanding Coordinates & Properties

CTS Mapper uses a 3D coordinate system to place objects in Stilwater. Understanding this is crucial for accurate placement:

X-Axis (Horizontal)

Left-to-right on the map. Negative values = west side, Positive values = east side of Stilwater.

Y-Axis (Depth)

Front-to-back on the map. Negative values = south, Positive values = north of the city.

Z-Axis (Vertical)

Height above ground. Typically 0-100+. Higher values = higher elevation (rooftops, elevated areas).

Rotation Values

0° = Facing East | 90° = Facing South | 180° = Facing West | 270° = Facing North. Use these to control which direction spawns or markers face.

Tips & Best Practices

Study the Map First

Spend time exploring the existing locations before making changes. Understand the layout and how locations relate to each other.

Use Landmarks

Reference major landmarks (bridges, water, district boundaries) when placing new locations to maintain logical world design.

Avoid Clipping

Don't place spawn points or markers inside buildings or underwater. Test in-game to verify good placement.

Backup Before Changes

Always backup your original CTS files before making extensive edits. Easy to revert if something breaks.

Test Incrementally

Test each location change in-game before making more edits. Easier to identify which change caused problems.

Document Coordinates

Keep notes of important coordinates you create. Useful for reference and sharing with other modders.

Respect District Boundaries

Saint's Row 2 has distinct districts. Maintain logical placement within appropriate neighborhoods.

Check Z-Axis Carefully

Height placement is crucial. Too low = underground, too high = floating in air. Match surrounding terrain.

Common Issues & Solutions

Spawn Points Don't Appear in Game

Cause: CTS data not properly repacked or incorrect file location.

Solution: Verify CTS files are in correct folder structure. Re-repack with ThomasJepp Tools.

Player Spawns Underground or in Building

Cause: Z-axis (height) value is too low or location is inside a building.

Solution: Increase Z value or move spawn point to open street. Test different coordinates.

Mission Won't Start at Custom Location

Cause: Spawn point not properly linked to mission in XTBL or script files.

Solution: Check mission data files. May need to edit additional files beyond CTS data.

CTS Mapper Won't Load Map Data

Cause: Extracted files are corrupted or in wrong location.

Solution: Re-extract fresh game files. Verify correct game version compatibility.

Markers Disappear After Repacking

Cause: CTS data saved in incompatible format or incorrect version.

Solution: Check CTS Mapper export format settings. Ensure you're using correct file format.
