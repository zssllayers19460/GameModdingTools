Tool Details & Usage

🔊 UnXWB

UnXWB is a user-friendly tool for extracting audio from XWB files. Simply drag and drop an XWB file onto the executable, and it will extract all audio tracks as WAV files into a folder named after the source file.

unxwb.exe radio_music.xwb

Extracts all audio to a folder named "radio_music" in the same directory.

🎵 UnWAV

Alternative extraction tool with additional format support. Works similarly to UnXWB but offers different extraction options.

unwav.exe voices.xwb

Extracts audio with additional format detection and conversion options.

⚡ XactExtract

Command-line utility for power users and batch processing. Perfect for extracting multiple XWB files automatically.

for %f in (*.xwb) do xactextract.exe "%f"

Batch extracts all XWB files in the current folder.

🛠️ XactTool

Repacks edited audio back into XWB format. Essential for completing the audio modding workflow.

xacttool.exe --build my_custom_audio.xwb

Builds a new XWB file from edited WAV files.

Creating Custom Radio Stations

One of the most popular uses for the Audio Tool Suite is creating custom radio stations. Here's how:

  1. Extract the radio station XWB files using UnXWB or XactExtract
  2. Identify the tracks you want to replace (they're usually named with track numbers)
  3. Edit your custom music to match the original format (22,050 Hz or 44,100 Hz, 16-bit WAV)
  4. Rename your custom tracks to match the original filenames
  5. Use XactTool to rebuild the XWB file with your custom tracks
  6. Repack into VPP_PC and test in-game

🎵 Radio Modding Tips

Keep track lengths similar to original songs for consistent timing. Normalize volume levels to match other radio stations. Create a backup of original radio files before replacing.

Audio Format Requirements

Format
WAV (Microsoft PCM)

Output Format

🎚️ Format Matching is Critical

Always match the original file's sample rate and bit depth when editing. Using mismatched settings can cause audio playback issues or crashes. Extract a reference file first to check the original specs using a tool like MediaInfo.

Pro Tips & Best Practices

🎧 Use Reference Files

Always keep original extracted files as references for format specifications. Compare your edits to ensure proper settings.

📊 Normalize Volume Levels

Ensure your custom audio matches the game's volume levels. Use normalization to prevent audio from being too loud or too quiet.

🔁 Test Incrementally

Replace one audio file at a time and test. This makes it easier to identify problematic files if something goes wrong.

💾 Backup Originals

Always keep backups of original XWB files. You'll need them if something goes wrong with your mod.

📁 Organize Your Project

Create separate folders for Original_XWB, Extracted_WAV, Edited_Audio, and Final_XWB to stay organized.

🎵 Custom Radio Station Names

You can't rename radio stations without editing game files, but you can replace the music entirely.

Troubleshooting

Audio doesn't play after repacking

Fix: Verify your WAV files match the original format specifications. Use MediaInfo to check sample rate, bit depth, and channels of the original files and match them exactly.

Game crashes when audio should play

Fix: Ensure your edited audio files aren't corrupted. Test with a single replacement before batch processing. Check that file sizes are reasonable.

Audio sounds distorted or sped up

Fix: Sample rate mismatch — ensure your edited files use the same sample rate as the original (usually 22,050 Hz or 44,100 Hz).

Tool won't extract files

Fix: Make sure the XWB file is not corrupted. Try a different extraction tool from the suite. Re-extract from VPP_PC if needed.

XactTool fails to build XWB

Fix: Check that all your WAV files are in the correct format. Ensure you have all required files present. Check for special characters in filenames.
