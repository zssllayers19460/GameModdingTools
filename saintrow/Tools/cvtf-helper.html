What is CVTF Helper?

CVTF Helper is a user-friendly GUI tool designed to simplify Saints Row 2 vehicle customization. Created by Masamaru, this tool acts as a front-end interface for the sr2_vehicle_table_crunch.exe utility, allowing you to process vehicle customization files with a simple drag & drop operation.

Instead of dealing with command-line parameters, CVTF Helper provides an intuitive graphical interface that makes vehicle modding accessible to everyone. Simply drag and drop your vehicle files onto the application, and it handles the rest.

⚠️ Important Note

CVTF Helper requires the sr2_vehicle_table_crunch.exe utility from the "Volition Alpha Tools" thread to function. You'll need to download this separately and place it in the same folder as CVTF Helper.

Troubleshooting

Tool doesn't do anything when dropping files

Fix: Ensure sr2_vehicle_table_crunch.exe is in the same folder as CVTF Helper.

Vehicle mods not showing in-game

Fix: Verify your files were processed correctly. Ensure your mod is installed in the correct game directory.

Game crashes with custom vehicles

Fix: Check that your vehicle files are compatible with the version of Saints Row 2 you're using.
