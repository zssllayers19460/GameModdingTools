What is the Model Submesh Tool?

The Model Submesh Tool (cmesh2obj_0-2_submeshes.exe) is an advanced version of the standard Model Export Tool that provides finer control over Saints Row 2 3D models. Created by Masamaru, this enhanced utility does everything the standard tool does — but with one critical addition: it splits complex models into individual submeshes.

Many Saints Row 2 models are composed of multiple submeshes — for example, a character model might have separate submeshes for the head, torso, arms, legs, and clothing items. This tool exports each submesh as a separate OBJ file, allowing you to edit specific components without affecting the whole model. This is essential for advanced modding tasks like removing specific clothing items, editing individual character parts, or isolating vehicle components.

🔧 What Are Submeshes?

Submeshes are individual components that make up a larger model. For example, a character might have separate submeshes for the head, shirt, pants, shoes, and accessories. The Submesh Tool exports each as its own OBJ file for precise editing.

Understanding Submesh Naming

When you export a model with the Submesh Tool, you'll receive multiple OBJ files with names that indicate what part of the model they represent. Common submesh names include:

  • Character Models: head, torso, arms, legs, hands, feet, hair, eyes, mouth, shirt, pants, shoes, jacket, hat, glasses, accessories
  • Vehicle Models: body, chassis, wheels, tires, windows, doors, hood, trunk, spoiler, lights, interior, engine
  • Weapon Models: barrel, grip, stock, magazine, scope, sights, body, accessories
  • Environment Models: base, roof, walls, windows, doors, details, decorations

🔍 Not Sure What a Submesh Is?

Open the OBJ file in Blender or a text editor to see what it contains. The geometry will give you clues about which part of the model it represents.

Advanced Editing Techniques

👕 Removing Clothing

Isolate clothing submeshes (shirt, pants, jacket) and delete or replace them to create different outfit variations.

🎨 Component Swapping

Export submeshes from different models and combine them — for example, take a shirt from one character and put it on another.

🚗 Vehicle Part Customization

Edit wheel submeshes separately to create custom rims, or modify body panels for unique vehicle designs.

🧩 Preserve Rigging

When editing character submeshes, maintain the original vertex order and weight maps to preserve animation rigging.

📐 Scale Consistency

Keep edited submeshes at the same scale as the original to ensure proper fit when reassembling.

🔧 Test Component by Component

Edit one submesh at a time and test in-game to identify issues before making many changes.

💡 Which Tool Should You Use?

Use the standard Model Export Tool for simple whole-model edits or when you're just starting with 3D modding. Use the Submesh Tool when you need component-level precision — like removing a specific piece of clothing or editing vehicle wheels separately.

Troubleshooting

Tool creates many OBJ files — which one do I edit?

Tip: Open OBJ files in a 3D viewer or import them one by one to identify which submesh corresponds to which part. Name them or make notes for reference.

Edited submesh doesn't fit with other parts after reassembly

Fix: Maintain the original scale and pivot point when editing. Import a reference submesh to ensure alignment.

Submesh names aren't descriptive

Tip: Some models have generic submesh names. Import them into Blender to see the geometry and rename files yourself for easier reference.

Game crashes with custom model after reassembly

Fix: Ensure all submeshes are present and properly reassembled. Test with a single edited submesh first before making multiple changes.

Can't find submesh files after export

Fix: Check the same folder as the source model file. Submesh files are saved there with descriptive names.
