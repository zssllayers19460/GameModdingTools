What is the PEG Tool?

The PEG Tool is a specialized utility for extracting, converting, and editing ".peg" texture files from Saints Row 2. PEG files are the game's proprietary texture format, and they're used everywhere—character skins, vehicle paintjobs, clothing, building textures, weapon skins, and more.

Think of it like this: ThomasJepp Tools extracts the assets, but the PEG Tool lets you edit the textures. If you want to recolor a character's outfit, change a vehicle's paint job, or create custom weapon skins, the PEG Tool is essential.

Core Function

Convert PEG texture files ↔ Standard image formats (PNG, TGA, DDS) → Edit in Photoshop/GIMP → Convert back to PEG → Game loads your custom texture

Key Features & Capabilities

Format Conversion

Convert PEG files to standard image formats (PNG, TGA, DDS) that can be edited in any image editor.

Batch Processing

Convert multiple PEG files at once, saving hours when working with large texture packs.

Reverse Conversion

Convert your edited images back to PEG format for the game to recognize and load them.

Mipmap Handling

Automatically processes mipmaps (texture LODs) to ensure textures look good at all distances.

Quality Preservation

Maintains texture quality and compression settings during conversion without quality loss.

Multi-Format Support

Works with various texture file formats and automatically handles color spaces.

Texture Editing Workflow

Here's the complete process for editing textures with the PEG Tool:

1

Extract Assets with ThomasJepp

First, use the ThomasJepp Saints Row Tools to extract the VPP_PC files and locate the PEG files you want to edit. They'll be organized in folders by asset type.

extracted_folder/textures/characters/player_skin.peg
extracted_folder/textures/vehicles/taxi_paint.peg
2

Open PEG Tool

Launch the PEG Tool application. You'll have options to convert PEG files to standard image formats or vice versa.

Tip: Create a working folder specifically for texture editing. Keep original PEGs in one folder and conversions in another.

3

Convert PEG to Editable Format

Select the PEG file and convert it to PNG, TGA, or DDS format. This creates an image file you can edit in standard image editors.

  • PEG → PNG (smaller file size, good for web/sharing)
  • PEG → TGA (lossless, best for editing)
  • PEG → DDS (DirectDraw Surface, native format for many engines)
4

Edit Texture in Image Editor

Open the converted image in your preferred image editor and make your modifications.

Popular Image Editors

  • Photoshop: Professional-grade, industry standard
  • GIMP: Free, open-source, very capable
  • Krita: Free, designed for digital painting
  • Aseprite: Great for pixel art and sprites
5

Save Your Edited Image

Save your modified texture back to the same format (PNG/TGA/DDS). Make sure to:

  • Maintain the same resolution as the original
  • Use lossless compression if available
  • Keep the file name consistent
  • Avoid RGB color space issues (use the same color space as original)
6

Convert Back to PEG Format

Use the PEG Tool to convert your edited image back to PEG format. Select the edited image and choose to convert to PEG.

Important: Match the original PEG's compression settings and resolution. The tool usually auto-detects this.

7

Replace Original PEG File

Copy your newly converted PEG file back into the extracted asset folder, replacing the original file. Maintain the exact same file path.

extracted_folder/textures/characters/player_skin.peg (REPLACED)
8

Repack with ThomasJepp

Use ThomasJepp Tools to repack your modified assets back into the VPP_PC archive file for the game to load.

9

Test in Game

Launch Saints Row 2 and verify your texture modifications appear correctly. Check for:

  • Correct colors and appearance
  • No texture stretching or distortion
  • Proper lighting and shading
  • No missing or corrupted textures

Texture Editing Tips & Best Practices

Match Resolution

Always match the original texture resolution. Upscaling can look blurry; downscaling loses detail.

Study Original Textures

Open the original texture in your editor to understand shading, highlights, and material properties before editing.

Use Reference Images

Find reference photos or artwork to base your custom textures on. This improves visual quality significantly.

Test Mipmaps

Your texture might look great zoomed in but weird from far away. Test at various distances in-game.

Lighting & Shadows

Pay attention to existing lighting in the texture. Match the light direction and intensity for consistency.

Save Originals

Always keep copies of the original textures. You'll want them as reference and for reverting if needed.

Color Correction

Avoid oversaturated colors. Saints Row 2 has a specific color palette—match it for best results.

Batch Convert

If editing multiple textures, batch convert them all at once to save time rather than one-by-one.

Common Issues & Solutions

Texture Won't Convert

Cause: Corrupted PEG file or unsupported format.

Solution: Verify the PEG file is intact. Try re-extracting from the original game files.

Edited Texture Looks Wrong in Game

Cause: Resolution mismatch, color space issues, or compression problems.

Solution: Ensure your edited image matches the original resolution and color space. Re-convert with PEG Tool.

Texture Colors Appear Washed Out

Cause: Color space conversion error or gamma mismatch.

Solution: Check your image editor's color space settings. Match the original texture's color profile.

Pink/Magenta Texture in Game

Cause: Missing or corrupted texture file.

Solution: Verify the repacked PEG file is in the correct location and properly compressed.

Tool Crashes on Conversion

Cause: Out of memory or incompatible file format.

Solution: Close other applications, try smaller textures first, ensure file format compatibility.
